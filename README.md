#Cli utility `ls`

**This is my first cli program in golang, by writing which I hope to get skills in developing cli programs in Go**

##Build and run

###Before build cd to `/cmd`

*On unix like*
```
go build main.go && ./main 
```

*On windows*  
```
go build main.go && ./main.exe 
```

## Run tests
```
go test -v
```
